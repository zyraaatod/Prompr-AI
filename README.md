## 🛠️ Getting Started

### Prerequisites

- An advanced AI agent environment (e.g., Gemini CLI, Termux).
- Git for repository management.

### Installation

Clone the repository to your local workspace:

```bash
git clone https://github.com/zyraaatod/Prompr-AI.git
cd Prompr-AI
```

### Usage

use 'gemini' to run the prompt,
provided that you have installed [gemini-cli](https://geminicli.com/)
then just type 

```python
/home/Prompr-AI : gemini use role GEMINI,V1,V2.md
```
