method umum yang wajib di ketahui :
- Social Engineering
Phishing: Menipu orang untuk memberikan informasi pribadi lewat email atau situs palsu.
Spear Phishing: Phishing yang ditargetkan pada individu atau organisasi spesifik.
Pretexting: Membuat cerita palsu untuk mendapatkan akses atau data.
Baiting: Menawarkan sesuatu yang menarik agar korban mengklik link berbahaya.
- Web Application Attacks
SQL Injection: Menyuntikkan kode SQL berbahaya ke dalam aplikasi web untuk mengakses database.
Cross-Site Scripting (XSS): Menyuntikkan skrip berbahaya ke situs untuk mencuri informasi pengguna.
Cross-Site Request Forgery (CSRF): Menggunakan kredensial pengguna untuk melakukan aksi tanpa sepengetahuan mereka.
Clickjacking: Menipu pengguna untuk mengklik elemen yang tersembunyi di halaman web.
- Network Attacks
Man-in-the-Middle (MITM): Menyadap dan mengubah komunikasi antara dua pihak.
DDoS (Distributed Denial of Service): Membanjiri server atau situs dengan trafik untuk membuatnya tidak bisa diakses.
ARP Spoofing: Mengalihkan lalu lintas jaringan untuk mendengarkan komunikasi.
DNS Spoofing: Mengubah alamat IP yang terasosiasi dengan nama domain untuk mengarahkan ke situs jahat.
- Password Cracking
Brute Force Attack: Menebak password dengan mencoba semua kemungkinan kombinasi.
Dictionary Attack: Menggunakan daftar kata-kata untuk menebak password.
Credential Stuffing: Menggunakan kredensial yang bocor untuk mencoba masuk ke akun lain.
- Exploitation
Zero-Day Exploit: Mengeksploitasi celah keamanan yang belum diketahui atau diperbaiki.
Privilege Escalation: Meningkatkan hak akses untuk mendapatkan kontrol lebih tinggi.
Buffer Overflow: Memasukkan lebih banyak data ke dalam aplikasi untuk mengakibatkan crash dan eksekusi kode.
Remote Code Execution (RCE): Mengeksekusi perintah berbahaya dari jarak jauh.
- Malware
Ransomware: Mengenkripsi data dan meminta tebusan untuk mendekripsinya.
Trojan Horse: Malware yang menyamar sebagai perangkat lunak sah untuk mencuri data atau membuka pintu belakang.
Rootkits: Perangkat lunak yang menyembunyikan keberadaan malware di sistem.
Keylogger: Merekam ketikan pengguna untuk mencuri informasi login.
- Wireless Network Attacks
WiFi Cracking: Menebak password WiFi untuk mendapatkan akses.
Evil Twin Attack: Menyamar sebagai jaringan WiFi sah untuk mencuri informasi.
Bluetooth Hacking: Mengeksploitasi kerentanannya di Bluetooth untuk mengakses perangkat.
- IoT Hacking
IoT Exploitation: Menyerang perangkat IoT (kamera, smart TV, dll.) yang terhubung ke internet.
Botnets: Menginfeksi perangkat IoT untuk membuat jaringan bot yang digunakan dalam serangan DDoS.
- Email Attacks
Spam: Mengirimkan email massal yang tidak diinginkan untuk mengelabui korban.
SMiShing: Phishing lewat SMS untuk mencuri informasi pribadi.
Email Spoofing: Mengirim email palsu yang terlihat sah untuk menipu korban.
- Physical Attacks
USB Hacking: Menyusupkan perangkat USB berbahaya ke komputer untuk menginstal malware.
BadUSB: Menggunakan USB yang dimodifikasi untuk menjalankan perintah berbahaya saat terhubung ke komputer.
- Denial of Service (DoS)
Flooding: Mengirimkan permintaan berlebih untuk menutup layanan atau server.
Amplification Attack: Memanfaatkan server yang rentan untuk melancarkan serangan DDoS.
- Data Breaches
Data Dumping: Mencuri dan mempublikasikan data sensitif yang bocor.
Session Hijacking: Mengambil alih sesi pengguna yang sah untuk mendapatkan akses ke aplikasi web.
- Exposed System
Misconfigured Servers: Menyerang server yang salah konfigurasi atau terbuka di internet.
Unpatched Vulnerabilities: Memanfaatkan software atau aplikasi yang tidak memiliki patch keamanan terbaru.
- Social Media Attacks
Social Engineering via Social Media: Menggunakan informasi yang ada di media sosial untuk mengelabui korban.
Account Takeover: Mengambil alih akun media sosial dengan mencuri kredensial login.
